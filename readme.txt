The following publicly available libraries are used in the project:
 + GLAD, generated for the OpenGL|ES 3.2 core (https://glad.dav1d.de/)
 + GLFW 3.3.8, API for creating windows, contexts and surfaces, getting input data and events (https://www.glfw.org/)	
 + stb_image 2.28, image file loader (https://github.com/nothings/stb/blob/master/stb_image.h)
 + GLM 0.9.9.7, mathematics library on the GLSL specifications (https://github.com/g-truc/glm)
	
The managment of Hue/Saturation/Brightness effects:
  Press to change the Hue:        	Q (dec) - W (reset) - E (inc)
  Press to change the Saturation: 	A (dec) - S (reset) - D (inc)
  Press to change the Brightness: 	Z (dec) - X (reset) - C (inc)

Press Esc to exit the program.

В проекте используются следующие общедоступные библиотеки:
 + GLAD, сгенерированный для ядра OpenGL|ES 3.2 (https://glad.dav1d.de/)
 + GLFW 3.3.8, API для создания окон, контекстов и поверхностей и обработки событий (https://www.glfw.org/)	
 + stb_image 2.28, загрузчик файлов-изображений (https://github.com/nothings/stb/blob/master/stb_image.h)
 + GLM 0.9.9.7, математическая библиотека на спецификации GLSL (https://github.com/g-truc/glm)	
	
Управление эффектами оттенка/насыщенности/яркости:	
  Нажмите для изменения Оттенка:	Q (уменьшить) - W (сброс) - E (увеличить)
  Нажмите для изменения Насыщености: 	A (уменьшить) - S (сброс) - D (увеличить)
  Нажмите для изменения Яркости:	Z (уменьшить) - X (сброс) - C (увеличить)

Нажмите Esc для выхода из программы.
